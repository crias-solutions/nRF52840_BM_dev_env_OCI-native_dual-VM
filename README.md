# nRF52840_BM_dev_env_OCI-native_dual-VM
Zero-cost cloud-native nRF52840 bare-metal development environment on OCI Always Free Tier. Dual-VM architecture (Build Server + Hardware Gateway) provisioned via Terraform. Features USB/IP tunneling for remote flashing and Claude Code Agentic integration. Includes ARM GCC, nRF5 SDK, nrfjprog, and JLink.
